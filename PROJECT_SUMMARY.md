# Project Summary

## Recommended repository name

**asl-gesture-recognition-cnn-transfer-learning**

This name is hiring-manager-friendly because it clearly signals ASL gesture recognition, CNN computer vision, transfer learning, and practical PyTorch implementation.

## Alternative repository names

- `pytorch-asl-gesture-classifier`
- `asl-hand-sign-recognition-cnn`
- `alexnet-transfer-learning-asl`
- `computer-vision-asl-gesture-recognition`

## GitHub short description

PyTorch ASL hand-gesture classifier using a custom CNN, hyperparameter tuning, and AlexNet transfer learning with held-out test evaluation.

## LinkedIn project description

Converted an ASL gesture-recognition lab into a modular PyTorch computer-vision project. The repository includes ImageFolder data loading, reproducible train/validation/test splitting, a custom CNN architecture, training and evaluation scripts, checkpointing, hyperparameter experiments, small-dataset overfit sanity checks, and transfer learning with pretrained AlexNet features.

The final transfer-learning approach improved test accuracy from approximately 79.9% with a custom CNN to approximately 94.0% by using pretrained convolutional features and a lightweight classifier head.

## Resume bullets

- Developed a PyTorch ASL hand-gesture recognition pipeline using custom CNNs and AlexNet transfer learning, with modular data loading, model training, checkpointing, hyperparameter tuning, and held-out test evaluation.
- Improved model generalization from ~79.9% custom-CNN test accuracy to ~94.0% using pretrained AlexNet feature extraction and a compact classifier head.
